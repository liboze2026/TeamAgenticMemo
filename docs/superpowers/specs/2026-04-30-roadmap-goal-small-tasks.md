# TeamAgent Roadmap Goal（Small Tasks）

> 日期：2026-04-30
> 来源：`2026-04-30-roadmap-goal-verification.md`（拆分）

## 最终目标（Goal）

用户无感的上传下载，分享踩坑经验和实用用法，团队里别人踩过的坑后人不再踩。

## 小任务拆分（Small Tasks）

### Task 1：跨人同步打通
**目标**：A 的经验可自动传给 B，并在 B 的后续会话里生效。

### Task 2：冲突识别与治理
**目标**：互斥经验可被识别，并有明确裁决结果（保留/合并/下线）。

### Task 3：隐私守门
**目标**：同步前能拦截或脱敏敏感信息（如邮箱、内部主机名、内部路径）。

### Task 4：反噬控制
**目标**：被同步的团队经验不会大量误伤，拒绝/回滚维持在可接受范围。

### Task 5：收益声明（先于验证）
**目标**：先完成可审计的收益声明（Claim），明确定义口径、假设、证据来源与反证条件；暂不把“重复犯错率相对基线下降 ≥ 30%”作为当前验收项。
**当前实现**：见 `docs/superpowers/specs/2026-04-30-task5-benefit-claim-v0.md`。

### Task 6：真实团队稳定运行
**目标**：≥ 3 人连续使用 ≥ 1 周，且 Task 1-5 持续成立。
