## 一、单句目标

> **在用户发送 prompt 时，通过 UserPromptSubmit hook 从 sqlite-vec 语义检索相关 wiki 条目，以指令式格式注入上下文，让 Claude 主动引用最新知识。**

---

## 二、背景与定位

M2.6 建立了 wiki 索引（Index 层）。M2.7 是检索注入层（Retrieve + Inject）。

```
M2.6 Index:    抓取 → 过滤 → LLM提炼 → 存SQLite → 生成向量
M2.7 Retrieve: UserPromptSubmit → 关键词提取 → sqlite-vec余弦匹配 → Top-K → 注入上下文
Generate:      Claude Code 本身（注入后由 AI 自主采纳）
```

对应 Phase 2 四通道的**通道 4**（UserPromptSubmit Wiki 注入）。

---

## 三、设计决策记录

| # | 决策点 | 选择 | 原因 |
|---|--------|------|------|
| D1 | 频控机制 | 时间窗口（`last_injected_at`） | UserPromptSubmit hook 无状态进程，无法感知 session 边界；时间窗等价于 session 频控，零额外文件 |
| D2 | 查询向量构建 | 先提取关键词再嵌入 | prompt 长时噪声多，关键词短字符串和 wiki 的 `tldr+keywords` 嵌入方式对齐，相似度更准 |
| D3 | 注入格式 | 指令式（含明确指令让 AI 加 "💡 Latest Wiki" 小节） | 提高采纳率，AI 更可能主动引用 |
| D4 | 架构 | WikiRetrieverPort 先行（方案 B） | 符合 M0 约束（新 Port 先写契约测试），与 M2.5-half ErrorSignalCollector 模式一致 |
| D5 | 频控参数 | cooldown=30min、session窗=60min、上限=15条 | 同条 wiki 30 分钟内不重复 ≈ "3 轮内不重复"；60 分钟内 15 条上限 ≈ per-session 上限 |

## 四、整体架构 + 数据流

```
UserPromptSubmit hook fires
         ↓
bin-user-prompt-submit.ts  (stdin JSON: { prompt: string })
         ↓
extractQueryKeywords(prompt)        ← Core 纯函数
  ├─ regex 抓 import/from/require 里的包名
  ├─ 抓 npm install / pip install 等安装命令的包名
  ├─ 抓 pkg@version 版本号模式里的包名
  └─ 空 → fallback: prompt 前 200 chars
         ↓  string[]
buildQueryText(keywords, promptFallback)  ← Core 纯函数
         ↓  "axios AbortSignal fetch"（或 fallback 文本）
XenovaEmbedder.embed([queryText])   ← 复用 M2.6 已有实现
         ↓  number[384]
WikiRetrieverPort.query({           ← 新 Port（D4）
  embedding, minSimilarity: 0.75,
  maxAgeDays: 90, maxResults: 3,
  now, cooldownMinutes: 30,
  sessionWindowMinutes: 60, sessionMaxInjections: 15
})
         ↓  WikiInjectionEntry[]（频控过滤已在内部完成）
WikiRetrieverPort.recordInjection() ← 更新 last_injected_at + inline_injection_count
         ↓
formatInjection(entries)            ← Core 纯函数
         ↓  指令式注入文本 or ""（无匹配）
stdout → Claude 接收注入上下文
```

## 五、WikiRetrieverPort 契约

### 接口定义

```typescript
// packages/ports/src/wiki-retriever.ts

export interface WikiInjectionEntry {
  knowledgeId: string;
  tldr: string;
  sourceType: string;
  publishedAt: Date;
  similarity: number;
}

export interface WikiQueryOptions {
  embedding: number[];
  minSimilarity: number;        // default 0.75
  maxAgeDays: number;           // default 90
  maxResults: number;           // default 3
  now: Date;                    // 注入 now 保证纯函数可测试
  cooldownMinutes: number;      // default 30（同条 wiki 冷却期）
  sessionWindowMinutes: number; // default 60（"session" 时间窗）
  sessionMaxInjections: number; // default 15（session 内上限）
}

export interface WikiRetrieverPort {
  query(opts: WikiQueryOptions): Promise<WikiInjectionEntry[]>;
  recordInjection(knowledgeIds: string[], now: Date): void;
}
```

### 契约测试要点（`wiki-retriever-contract.ts`）

1. `query()` 返回数组，长度 ≤ `maxResults`
2. 结果按 `similarity` 降序排列
3. `user_thumbs_down = 1` 的条目不返回
4. `publishedAt` 超过 `maxAgeDays` 的条目不返回
5. `last_injected_at` 在 `cooldownMinutes` 内的条目不返回
6. 过去 `sessionWindowMinutes` 内注入总数 ≥ `sessionMaxInjections` → 返回空数组
7. `recordInjection()` 后，相应条目在冷却期内不再返回

### 核心查询

```typescript
// Step 1: session 总量检查（超限直接返回空）
const sessionCount = db.prepare(`
  SELECT COUNT(*) AS n FROM wiki_meta
  WHERE last_injected_at > datetime(?, '-' || ? || ' minutes')
`).get(now.toISOString(), opts.sessionWindowMinutes) as { n: number };

if (sessionCount.n >= opts.sessionMaxInjections) return [];

// Step 2: sqlite-vec 余弦搜索 + 多条件过滤
const rows = db.prepare(`
  SELECT
    wm.knowledge_id,
    wm.tldr,
    wm.source_type,
    wm.published_at,
    (1 - vec_distance_cosine(kv.embedding, ?)) AS similarity
  FROM knowledge_vec kv
  JOIN wiki_meta wm ON kv.knowledge_id = wm.knowledge_id
  WHERE
    wm.user_thumbs_down = 0
    AND (wm.last_injected_at IS NULL
         OR wm.last_injected_at < datetime(?, '-' || ? || ' minutes'))
    AND wm.published_at > datetime(?, '-' || ? || ' days')
    AND (1 - vec_distance_cosine(kv.embedding, ?)) >= ?
  ORDER BY similarity DESC
  LIMIT ?
`).all(
  embeddingBlob,
  now.toISOString(), opts.cooldownMinutes,
  now.toISOString(), opts.maxAgeDays,
  embeddingBlob, opts.minSimilarity,
  opts.maxResults
);
```

### recordInjection

```typescript
recordInjection(knowledgeIds: string[], now: Date): void {
  const stmt = db.prepare(`
    UPDATE wiki_meta
    SET last_injected_at = ?,
        inline_injection_count = inline_injection_count + 1
    WHERE knowledge_id = ?
  `);
  for (const id of knowledgeIds) {
    stmt.run(now.toISOString(), id);
  }
}

## 七、Core 纯函数（`packages/core/src/wiki-injection/`）

### `extractQueryKeywords`

```typescript
export function extractQueryKeywords(prompt: string): string[] {
  const keywords = new Set<string>();

  // import/from/require 语句里的包名
  const importRe = /(?:from|require|import)\s+['"](@?[\w\-\/\.]+)['"]/g;
  for (const m of prompt.matchAll(importRe)) {
    keywords.add(m[1].replace(/^@/, "").split("/")[0]);
  }

  // npm/pip/pnpm install 命令里的包名
  const installRe = /(?:npm install|yarn add|pnpm add|pip install|go get)\s+([\w\-@\/\.]+)/g;
  for (const m of prompt.matchAll(installRe)) {
    keywords.add(m[1].replace(/^@/, "").split("/")[0]);
  }

  // pkg@version 模式（axios@1.x → "axios"）
  const versionRe = /([\w\-]+)@[\d\.]+/g;
  for (const m of prompt.matchAll(versionRe)) keywords.add(m[1]);

  return [...keywords];
}

export function buildQueryText(keywords: string[], promptFallback: string): string {
  if (keywords.length === 0) return promptFallback.slice(0, 200);
  return keywords.join(" ");
}
```

**设计意图**：只抓明确出现的技术标识符，不做 NLP。宁漏勿噪。

### `shouldInject`（供单元测试）

```typescript
export interface FreqState {
  lastInjectedAt: Date | null;
  recentSessionCount: number;  // 过去 sessionWindowMinutes 内已注入总数
}

export interface FreqConfig {
  cooldownMinutes: number;
  sessionWindowMinutes: number;
  sessionMaxInjections: number;
  now: Date;
}

export function shouldInject(state: FreqState, config: FreqConfig): boolean {
  if (state.recentSessionCount >= config.sessionMaxInjections) return false;
  if (state.lastInjectedAt === null) return true;
  const minutesSince =
    (config.now.getTime() - state.lastInjectedAt.getTime()) / 60_000;
  return minutesSince >= config.cooldownMinutes;
}
```

### `formatInjection`

```typescript
// 最小接口，不依赖 ports 层（Core 禁止 import ports）
export interface WikiHint {
  tldr: string;
  sourceType: string;
  publishedAt: Date;
}

export function formatInjection(entries: WikiHint[]): string {
  if (entries.length === 0) return "";

  const lines = entries.map((e) => {
    const month = e.publishedAt.toISOString().slice(0, 7); // "2026-03"
    return `  - ${e.tldr} (${e.sourceType} ${month})`;
  });

  return [
    "---",
    "📚 [WIKI HINT — 若本次回答涉及以下主题，请在末尾添加",
    '    "💡 Latest Wiki" 小节（每条 1-2 行，引用来源）：]',
    ...lines,
    "---",
  ].join("\n");
}
```

`WikiInjectionEntry`（来自 Ports）满足 `WikiHint` 的结构，bin 层直接传入，无需转换。
```

输出样例：
```
---
📚 [WIKI HINT — 若本次回答涉及以下主题，请在末尾添加
    "💡 Latest Wiki" 小节（每条 1-2 行，引用来源）：]
  - axios 0.30: 新增 AbortSignal 原生支持 (npm 2026-03)
  - Zustand v5: createStore API 破坏性变更 (github_release 2026-01)
---
```

无匹配 → 空字符串，Claude Code 不注入任何内容。

## 八、bin 入口

```typescript
// packages/cli/src/bin-user-prompt-submit.ts
#!/usr/bin/env node
/**
 * UserPromptSubmit Hook 入口 (M2.7)
 * stdin JSON → keyword extract → embed → sqlite-vec query → stdout injection
 * 任何异常 exit 0（不阻断用户输入）
 */
import os from "node:os";
import path from "node:path";
import {
  extractQueryKeywords,
  buildQueryText,
  formatInjection,
} from "@teamagent/core";
import {
  XenovaEmbedder,
  SqliteWikiRetriever,
  openDb,
  normalizeCwd,
} from "@teamagent/adapters";

const DEFAULT_FREQ = {
  cooldownMinutes: 30,
  sessionWindowMinutes: 60,
  sessionMaxInjections: 15,
};

async function main() {
  const chunks: Buffer[] = [];
  for await (const chunk of process.stdin) chunks.push(chunk as Buffer);
  const input = JSON.parse(Buffer.concat(chunks).toString()) as {
    prompt: string;
  };

  const cwd = normalizeCwd(process.env["CLAUDE_PROJECT_DIR"] ?? process.cwd());
  const dbPath = path.join(cwd, ".teamagent", "knowledge.db");

  const db = openDb(dbPath);
  const embedder = new XenovaEmbedder();
  const retriever = new SqliteWikiRetriever(db);
  const now = new Date();

  const keywords = extractQueryKeywords(input.prompt);
  const queryText = buildQueryText(keywords, input.prompt);
  const [embedding] = await embedder.embed([queryText]);

  const entries = await retriever.query({
    embedding,
    minSimilarity: 0.75,
    maxAgeDays: 90,
    maxResults: 3,
    now,
    ...DEFAULT_FREQ,
  });

  if (entries.length > 0) {
    retriever.recordInjection(
      entries.map((e) => e.knowledgeId),
      now,
    );
  }

  process.stdout.write(formatInjection(entries));
}

main().catch(() => process.exit(0));
```
