## 十五、价值主张（回到"为什么这个系统值得做"）

### 对开发者个人

- **同一个坑不再踩** — 我上周被纠正过一次，AI 下次就学会了。
- **经验不再丢** — 我今晚写总结了一条"在 core/ 下不能 import node:fs"，明天早上另一个项目的 AI 也会遵守。
- **认知负担降低** — 不用手写维护 CLAUDE.md，系统自动从纠正里抽规则。

### 对团队（Phase 4 兑现后）

- **资深工程师的经验自动扩散** — 不用写 onboarding 文档、不用开分享会。
- **新人第一天上手** — Claude Code 已经带着团队 50 条最佳实践。
- **集体踩坑 → 集体免疫** — 团队任何人踩过一次，所有人的 AI 马上知道。

### 对组织（Phase 5-6）

- **知识资产沉淀** — 团队的技术选型、工作流、约定全部可查、可审计、可回滚。
- **AI 工具无关** — 通过 MCP 标准支持 Cursor/Windsurf/... 不绑定单一工具。
- **合规友好** — 全本地、零云依赖，知识不外传。

### 市场差异化

**重复一次**：在"自动捕获规则 × 实时拦截"的象限里，**市场目前没有竞品**。Microsoft AGT、Codacy 在手写规则一侧；Anthropic Auto-Memory、claude-mem 在事后参考一侧。TeamAgent 是唯一全覆盖象限。
