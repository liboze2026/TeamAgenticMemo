# Test Plan
Generated by /plan-eng-review on 2026-04-26
Branch: main
Repo: V4teamBrain plus insights-share plugin

## Affected Pages/Routes
- CLI and hook flow: `capture-async.sh` -> `finalize-buffer.sh` -> `sync-mirror.sh` -> `inject-insights.sh` -> `rate-lesson.sh`.
- Mirror repo artifacts: `raw/<session>.jsonl`, `lessons.jsonl`, `ratings.jsonl`, `gate1/week-N.md`, `gate1/artifacts/*.jsonl`.
- Gate 1 operator commands: preflight, weekly summary, PII sample, latency sample, usage ledger, retention purge.

## Key Interactions to Verify
- Four engineers pass preflight before Gate 1 starts: plugin version, hook wiring, PAT validation, mirror clone, local cache, write test.
- A source session produces a lesson, a receiving session gets it injected, and `/insight-rate good` writes a joinable rating record.
- Cross-engineer is computed from hashed git identity, not raw email and not Unix user.
- Raw upload writes only post-filter payloads allowed by the chosen Gate 1 design and is sampled by the weekly PII audit.
- Weekly memo generation reads ratings, usage, latency, privacy audit, and incident fields into `gate1/week-N.md`.

## Edge Cases
- Missing PAT or mirror clone blocks preflight and prevents the four-week timer from starting.
- Same engineer across two sessions does not count as cross-engineer.
- Different engineers with same Unix username still count as different when git identity hashes differ.
- PII leak in sampled raw upload triggers kill condition and retention purge path.
- `claudefast` timeout or low confidence records usage and dropped-turn counts.
- Raw retention purge removes or quarantines raw files older than 30 days while preserving filtered weekly reports.

## Critical Paths
- End-to-end Gate 1 evidence chain: source lesson -> raw upload -> lesson metadata -> injection -> rating -> weekly summary -> week-4 pass/fail.
- Privacy kill path: sampled raw record leaks PII -> Gate 1 stops -> Stop hook disabled -> raw purge/retention path runs -> failure memo written.
- Performance kill path: UserPromptSubmit p95 exceeds 1000ms -> Gate 1 stops with evidence.
